# Smoke tests

These are lightweight, dependency-minimal sanity checks for core functionality.
They are *not* intended to be exhaustive unit tests.

## Run

From the repo root (with the venv activated or using `.venv/bin/python`):

- `./.venv/bin/python tests/smoke/test_metrics_smoke.py`
- `./.venv/bin/python tests/smoke/test_fista_smoke.py`
